# Vim config

Our main vim settings, using Vundle to manage various plugins

## Installation

### Public use

`git clone git://github.com/jackjoe/vim.git ~/.vim`

### Private use

`git clone git@github.com:jackjoe/vim.git ~/.vim`

### Run

```
$ ./install
```

For your local settings, just create a _vimrc.local_ file
